# Book List API

This project is a FastAPI-based REST API for managing a list of books.

## Project Structure
book-list/
├── main.py
├── requirements.txt
└── README.md

## Installation
pip install -r requirements.txt

## Running the Server
uvicorn main:app --reload

## API Documentation
http://127.0.0.1:8000/docs
